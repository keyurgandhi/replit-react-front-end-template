
# 🚀 React Frontend Template

Reusable React template using Vite + Axios + Env config.

## ✅ Setup

```bash
npm install
npm run dev
```

Create `.env` file:

```
VITE_API_BASE_URL=https://jsonplaceholder.typicode.com
```

## ✅ SAFE TO MODIFY
- src/pages
- UI

## ❌ DO NOT MODIFY
- package.json
- vite.config.js
- src/api

## ✅ API usage

```js
getUsers()
```
